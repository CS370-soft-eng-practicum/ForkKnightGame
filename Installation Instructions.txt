\f0\b\fs24 \cf0 Installation Instructions: \
\pard\pardeftab720

\b0 \cf2 1. Make sure you are using the latest webplayer plugin from: {\field{\*\fldinst{HYPERLINK "http://unity3d.com/webplayer/"}}{\fldrslt \cf3 http://unity3d.com/webplayer/}}\cf3  \cf2  and that you are also using PC or Mac. \cf3 \
\
2.  \cf2 When building a 
\b \cf2 Web Player
\b0 \cf2 , Unity automatically generates an HTML file next to the player data file. It contains the default HTML code to load the web player data file. Your HTML file is  "yourgame.html". Your player data is "yourgame.unity3d".\
\
3. Upload both 
\b \cf2 .html
\b0 \cf2  and 
\b \cf2 .unity3d
\b0 \cf2  to your server\
\
*** You may have to change the directory of unity3d file with in the body of the html doc. So if you have uploaded your .unity3d file to 
\b \cf2 files/themes/yourgame.unity3d
\b0 \cf2  then change the directory in within the body of your html file.\
\

\b \cf2 http://docs.unity3d.com/Documentation/Manual/HTMLcodetoloadUnityWebPlayercontent.html
